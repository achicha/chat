Async Client-Server chat written in python.
=================


Quick start
-------
.. code::

   pip install aiogbchat

Documentation:
-------
`<https://achicha.github.io/chat/>`_


How to generate docs:
-------

.. code::

   pip install sphinx
   sphinx-apidoc -f ../../chat -o /some_dir/docs/source
   make html
